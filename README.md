# Duarte Car — Site One-Page

Arquivos gerados:

- `index.html` — página única do site.
- `styles.css` — estilos seguindo paleta (vermelho, preto, branco, cinza).
- `script.js` — interações: abrir WhatsApp e Traçar rota no Google Maps.

Instruções rápidas:

1. Coloque a logomarca enviada como `DC-doP.jpg` dentro da pasta `assets` (crie `assets` na raiz do projeto).
2. Opcional: adicione uma imagem de fundo chamada `hero.jpg` em `assets` para o visual do hero (caso não exista, o fundo ficará escuro).
3. Abra `index.html` no navegador (duplo clique ou usar um servidor estático) para visualizar.
